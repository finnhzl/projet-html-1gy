Supports de stockage

introduction

Supports magnétiques : disque dur, disque ssd, disque Blu Ray ​ -->Le stockage magnétique ou enregistrement magnétique est le stockage de données sur des supports magnétisés. Le stockage magnétique utilise différents modèles de magnétisation dans des matériaux magnétisables pour stocker des données et est un type de mémoire non volatile. Accédez aux informations à l'aide d'une ou plusieurs têtes de lecture/écriture.​ Mémoire flash : clé usb, carte mémoire​ --> Si on veut faire très simple, la mémoire flash est avant tout un moyen de stockage de données informatiques (photos, musiques, vidéos, etc.). Plus précisément, il s'agit d'une mémoire de masse, c'est-à-dire d'un dispositif de stockage qui conserve les informations enregistrées même en l'absence d'alimentation. Cela signifie que même si vous éteignez votre ordinateur ou votre smartphone, vos données existent toujours.​ Stockage optique : DVD, CD​ --> La méthode de lecture d'un disque optique est similaire à celle utilisée sur un disque dur. Le disque est balayé par une tête de lecture, qui balaye les pistes du disque. Lorsque le disque est introduit dans le lecteur de carte, ce dernier commence à tourner sous l'action du moteur.​

  • élément 1 : Stockage de masse​ C'est quoi ? ​ --> c'est quand on stocke de nombreuses quantités de données​ ​ Son rôle ?​ Stocker les informations à long terme et en grande quantité​ ​

  • élement 2: C'est quoi la différence entre le stockage de masse et la mémoire RAM ?​ La RAM vide sa mémoire après que l'ordinateur est éteint tandis que le stockage garde toutes ses données jusqu'à ce qu'elles soient supprimées manuellement.​
